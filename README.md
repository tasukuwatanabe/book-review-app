# ブックレビュー管理アプリ

このプロジェクトは、フロントエンドライブラリの学習を目的としたブックレビュー管理アプリです。

## 使用技術スタック

| 分類             | 技術                             |
|------------------|----------------------------------|
| フレームワーク   | React                            |
| 言語             | TypeScript                       |
| スタイリング     | TailwindCSS                      |
| ルーティング     | TanStack Router                  |
| データ取得       | TanStack Query                   |
| バリデーション   | Zod                              |
| テスト           | Vitest, React Testing Library, MSW |
| ビルドツール     | Vite                             |

## セットアップ手順

以下の手順でローカル環境にプロジェクトをセットアップできます。

### 1. 依存関係のインストール

```bash
npm install
# または
yarn install
```

### 2. 開発サーバーの起動

```bash
npm run dev
# または
yarn dev
```
